# Find-Mine-Game
대학교에서 자바 스크립트 수업을 이수 중인 1학년 학생들을 대상으로 학기 말에 진행된 과제입니다.

> ### 실행

프로그램을 실행하면 지뢰가 묻힌 필드가 생성되고 출력됩니다.


실행 화면은 아래와 같습니다.


<img src="지뢰찾기 게임 사진/스크린샷 2026-06-07 155034.png" width="300">


> ### 깃발

지뢰 찾기를 진행하다보면 지뢰가 있을 법한 곳을 짐작할 수 있습니다. 그러면 다음 단계를 진행하기 위해 지뢰가 있다고 짐작되는 곳의 암기가 요구되는데 이러한 불편을 해소하기 위해 지뢰찾기 게임에는 '깃발' 기능이 있습니다.


지뢰가 존재하지 않는 곳을 파헤칠 수 있고 지뢰가 있다고 추정되는 곳에는 깃발을 꼽아 표시하는 것입니다.


f 행 열 을 입력해 깃발을 꼽을 수 있습니다. 깃발을 꼽으려는 위치가 이미 열린 셀인 경우 안내 메시지를 출력합니다.


실행 화면은 아래와 같습니다.

> ### 게임 오버

지뢰찾기는 지뢰가 설치되어있는 땅을 파헤치면 패배하는 게임입니다.


지뢰를 밟아 패배하면 설치되어있던 지뢰의 위치를 전부 공개하며 패배 메시지를 출력합니다.

> ### 승리

지뢰찾기는 지뢰가 설치되어있는 곳과 그렇지 않은 곳을 완벽하게 구분하면 승리하는 게임입니다.


찾느라 힘들었습니다. 승리하면 메시지와 함께 승리를 만끽할 수 있습니다.
